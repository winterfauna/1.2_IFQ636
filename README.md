This is an implementation of a conference registration platform.
In order to access, the ec2 instance may need to be restarted, and the public IP address may need to be reaccquired from EC2, and updated in axiosConfig.jsx.

To run locally, uncomment the local host in axiosConfig, and comment the live host
Then, run npm start from root.

To run live, ensure live host is uncommented, and local host is commented in axiosConfig.jsx, then git push the updates.
Once push and build is complete, it will be available at the ip address listed.
